# UV

UV еқҗж Үдёәд»Һе·ҰдёҠеҲ°еҸідёӢпјҢжҳ е°„еҲ°д»ҺпјҲ0пјҢ0пјүеҲ°пјҲ1пјҢ1пјү

# е®һзҺ°жҖқжғі

еҜ№дәҺж–№еҪўжёҗеҸҳжҲ‘д»¬йҖҡиҝҮеҸ–зү№еҖјж–№дҫҝзҗҶи§Је’Ңи®ҫи®Ў
еҜ№дәҺиҙҙеӣҫй»‘зҡ„ең°ж–№дёә 0пјҢзҷҪзҡ„ең°ж–№дёә 1пјҢзҒ°зҡ„ең°ж–№дёә 0.5
еңЁеҜ№зӣ®ж ҮиҝӣиЎҢж°ҙе№іе’Ңз«–зӣҙж–№еҗ‘зҡ„еҲҶи§Ј
е°ұеҫҲе®№жҳ“еҫ—еҮәдёӨдёӘеҗ‘йҮҸ
жҲ‘д»¬жүҖйңҖиҰҒеҒҡзҡ„е°ұжҳҜжүҫеҲ°дёҖдёӘеҸҳжҚўпјҢдҪҝеҫ—дёҖдёӘеҗ‘йҮҸеҸҳдёәеҸҰдёҖдёӘеҗ‘йҮҸ
