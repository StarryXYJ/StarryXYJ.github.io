# 物理

动态平衡的专题结束了
感觉我牛二动力学还可以，就浅浅看了一点，之后做题再发现以下问题吧
另外是曲线运动（平抛）开了个头
动态平衡的总结的话
就是
先分析问题
多物体时，如果问题不涉及内力就用整体法，或者先用整体法把不涉及内力的问题解决，反正如果涉及内力基本上都要隔离
然后多物体隔离分析的话，先分析受力少的，一般都三个力
这样都化为单物体三力三笠动态分析
三力分析就三角形

重力恒不变

# 仅一力变向

# 二力变向

反向延长 + 平移
相似三角形

# 二力变向，夹角不变
